# Weather Data Analysis & Forecasting

## Project Overview
This project analyses historical weather data and predicts future weather trends using Machine Learning and Time Series Forecasting techniques.

The project uses the Prophet forecasting model to analyse temperature patterns and forecast future weather conditions.

---

## Objectives
- Analyse historical weather data
- Identify weather trends and seasonality
- Visualise temperature and rainfall patterns
- Forecast future temperature values
- Generate graphical insights

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Prophet
- Jupyter Notebook

---

## Dataset
The dataset contains historical weather information of Bengaluru city including:
- Date
- Temperature
- Rainfall
- Humidity

---

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Seasonality Analysis
5. Forecasting using Prophet
6. Visualization of Results
7. Future Prediction

---

## Features
- Temperature trend analysis
- Rainfall analysis
- Seasonal decomposition
- 30-day weather forecasting
- Forecast graph with confidence interval
